# ReadMe

Basic AWS Cognito Login with Email using Python boto3.

## Deployment
1. sam build; sam deploy
2. Get outputs
3. `sam build`
4. Use `sam deploy --guided` and replace with your own values
5. Delete the `parameter_overrides` in the `samconfig.toml` file
